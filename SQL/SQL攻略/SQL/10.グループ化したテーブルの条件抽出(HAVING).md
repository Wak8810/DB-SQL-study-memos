## HAVINGの使い方
- GROUP BYと一緒に使うのは必須
- WHEREはGROUP BYの前に条件
- HAVINGはGROUP BYの後に条件
- GROUPBYの後ろに条件式と共に記載
```sql
GROUP BY JOB
HAVING AVG(SAL) >= 2500
```

```Question
EMP表からJOB（仕事）でグループ化し、SAL（給料）の最大値が2000以上のJOBとSAL（最小、最大、平均）を表示してください。
```
```SQL
SELECT JOB,MIN(SAL),MAX(SAL),AVG(SAL)
FROM EMP
GROUP BY JOB
HAVING MAX(SAL) >= 2000
```
