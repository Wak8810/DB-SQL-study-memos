## 条件結合
- 複数の表にも対応
- SELECTやWHEREでcolumnを書くとき
	- table.columnみたいに書く
- FROM後ろでカンマ区切り
```sql
SELECT EMP.EMPNO,EMP.ENAME,DEPT.DEPTNO,DEPT.DNAME
FROM EMP,DEPT
WHERE EMP.DEPTNO = DEPT.DEPTNO
```
- WHEREなしでもいいけど、全部のカラム同士を合わせたものが出来る？
	- a,b,cと1,2,3で、a1a2a3b1b2b3みたいな
- 条件を加える
	- WHEREの後ろに、AND EMP.SAL>=1500
