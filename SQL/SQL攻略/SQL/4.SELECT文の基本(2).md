## WHERE句を使ったレコード選択の仕方
- whereの後ろに条件式を記述して、条件に合致するレコードのみを選択できる。
	- 使用可能なのは、=と><が使えるやつ
	- <>が、!=と同じ意味
```sql
SELECT ENAME,JOB,SAL 
FROM EMP
WHERE SAL >= 2000 
```
## 複合条件によるレコード選択
- 論理演算子も使用可能。
	- AND,OR,NOT。（）を使えば、優先順位を付けられる。
```sql
SELECT ENAME,JOB,SAL
FROM EMP
WHERE JOB='CLERK' AND SAL >= 1000
```
