## ORDER BY
- 指定した値を基準に並べ替え
- ORDER BY 列 並び順
	- 複数指定可能
	- 列 並び順をカンマでつなぐ
	- 列を列番号にしてもいいけど、あんま使わない
- 存在しない場合、順序が保障されないので注意
## OFFSET-FETCH
- 並べ替えた結果の一部を取得する
- ORDER BYの後につなげる
	- 使えないのもあり、Postgreは使えないらしい
	- oracleとか使えるね
```sql
SELECT * FROM table
ORDER BY column
OFFSET 除外する行数 ROWS
FETCH NEXT 取得行数 ROWS ONLY
```
- DBMSによっては、OFFSETも省略可能。
- FETCH省略したら、該当行全て
- 10,5で指定したら、11~15行ゲト
